# 🏦 Bank Management System

A console-based Bank Management System developed in Python using Object-Oriented Programming (OOP). This project allows users to manage bank accounts through a simple menu-driven interface.

## 🚀 Features

- Create a New Account
- Search Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- View All Accounts
- Delete Account
- Menu-Driven Interface

## 🛠️ Technologies Used

- Python
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the Python file.
4. Follow the menu instructions.

## 📚 Concepts Used

- Classes & Objects
- Constructor (`__init__`)
- Methods
- Lists
- Loops
- Conditional Statements
- Functions
- Object References (`self`)
